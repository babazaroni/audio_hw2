USB Audio Solution Overview

Functionality

Provides USB interface to audio I/O.

Supported Standards

USB

USB 2.0 (Full-speed and High-speed)

USB Audio Class 1.0

USB Audio Class 2.0

USB Firmware Upgrade (DFU) 1.1

USB Midi Device Class 1.0

Audio

I2S

S/PDIF

ADAT

Direct Stream Digital (DSD)

MIDI

Supported Sample Frequencies

44.1kHz, 48kHz, 88.2kHz, 96kHz, 176.4kHz, 192kHz, 352.8kHz, 384kHz

Supported Devices

XMOS Devices

XS1 L-Series

XS1 U-Series

XS1 G-Series (Not recommended for new designs)

Requirements

Development Tools

xTIMEcomposer Development Tools v13 or later

USB

External ULPI USB Phy (If using XS1 G/L-Series)

Audio

External audio DAC/ADC/CODECs (and required supporting componentry) supporting I2S

Boot/Storage

Compatible SPI Flash device
